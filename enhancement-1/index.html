Enhancement 1: Software Design and Engineering

This enhancement modularized the Animal Shelter Dashboard codebase and improved UI/UX design to create a professional, scalable application.

Key Features:

  • Modularized code into separate files for easier maintenance:
    • app.py: Main application file
    • callbacks.py: Interactive functions (data filtering, map updates)
    • layouts.py: User interface layout
    • data_processing.py: Data retrieval and processing
  • Flask server integration for scalability
  • UI/UX improvements for better navigation, responsiveness, and aesthetics
